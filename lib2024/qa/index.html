  14. Ques: Create CSS keyframes block called “spin” that rotates an element.

    Ans: @keyframes spin {
    from {
    transform: rotate(0deg);
    }
    to {
    transform: rotate(360deg);
    }
    }

    animation-duration: 0.5s;
    animation-iteration-count: 5;
    animation-timing-function: 0.5s;
    }
    @keyframes spin {
